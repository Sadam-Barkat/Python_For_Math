## What are Variables?
Containers for storing data values in Python.

## Types and Examples
- **int**: `age = 25`
- **float**: `pi = 3.14`
- **str**: `name = "Alice"`
- **bool**: `is_student = True`

## Naming Rules
- Start with a letter or `_`
- No spaces or special characters
- Avoid reserved keywords (`if`, `class`)
